## Поддерживаемые источники

> Midas принимает любой поток информации, который вы добровольно ему скормите. **Повторение — это и есть сигнал** — если вы встретили это более одного раза, скормите это Midas.

| Источник | Лучшая линза | Примечания |
|----------|:------------:|------------|
| Slack / Teams сообщения | Пробел спроса, Денежный сигнал | Командные каналы = фрустрации с поставщиками, бюджетные сигналы, ручные обходы |
| WhatsApp / iMessage / WeChat чаты | Пробел спроса, Сетевой путь | Повторяющиеся жалобы от разных людей = неудовлетворённый спрос |
| Фотоплёнка телефона (без фильтрации) | Арбитраж, Пробел спроса | Стройки, пустые витрины, толпы, цены, вывески |
| История просмотра YouTube / TikTok | Мост навыков, Поведение | Последовательный тематический фокус = скрытое накопление экспертизы |
| История браузера | Мост навыков, Денежный сигнал | Паттерны исследования, сравнение цен, путешествия решения проблем |
| История покупок / подписки | Денежный сигнал, Поведение | Повторные покупки = валидированный спрос, привычки на автопилоте |
| Заметки совещаний (внутренние) | Денежный сигнал, Пробел спроса | Бюджетные разговоры, жалобы на инструменты, сломанные процессы |
| Почта `.eml` / `.mbox` | Денежный сигнал, Сетевой путь | Комиссионные предложения, цепочки рекомендаций, котировки поставщиков |
| Чеки / подтверждения заказов | Денежный сигнал, Арбитраж | Самый честный сигнал того, что вы на самом деле цените |
| Вставить текст напрямую | Все 6 линз | Даже расшифровка одной голосовой заметки даёт сигналы |

См. [references/sources-guide.md](references/sources-guide.md) для полной иерархии источников и правил триангуляции.

## Установка

### Claude Code

> **Важно**: Claude Code ищет скиллы в `.claude/skills/` в **корне git-репозитория**. Запустите это в правильном месте.

```bash
# Установка в текущий проект (запускать в корне git-репозитория)
mkdir -p .claude/skills
git clone https://github.com/realteamprinz/midas-skill .claude/skills/midas-skill

# Или установить глобально (доступно во всех проектах)
git clone https://github.com/realteamprinz/midas-skill ~/.claude/skills/midas-skill
```

### OpenClaw

```bash
git clone https://github.com/realteamprinz/midas-skill ~/.openclaw/workspace/skills/midas-skill
```

### Зависимости (опционально)

Midas — это чистый Markdown — без runtime-зависимостей. Python-инструменты для автоматизированного сбора входных данных (парсер истории браузера, OCR чеков и т. д.) запланированы и будут жить в `tools/` в будущем релизе.

## Использование

В Claude Code или OpenClaw введите любую из триггерных фраз Midas:

```
Midas, mine this                 → вставьте любой ввод, получите сигнальный отчёт
Turn this into gold              → то же самое
What money signal is here        → более конкретная формулировка
点石成金                         → то же самое, на китайском
Midas, analyze this deal         → режим анализатора сделок
What am I missing                → скормите неделю своих сообщений
```

Midas прогоняет каждый ввод через 6-линзовый движок извлечения, перекрёстно сверяет с вашим логом `evolution/evolution.jsonl` и возвращает ранжированный Сигнальный отчёт с **Немедленными следующими действиями**.

### 6 линз

| # | Линза | Главный вопрос |
|---|-------|----------------|
| 1 | **Денежный сигнал** | Где деньги движутся, застревают или утекают? |
| 2 | **Пробел спроса** | Чего хотят люди, что никто не предоставляет? |
| 3 | **Окно арбитража** | Где ценность оценивается неправильно? |
| 4 | **Мост навыков** | Что вы уже знаете, что имеет рыночную стоимость? |
| 5 | **Сетевой путь** | Кто с кем должен поговорить, и какова комиссия посредника? |
| 6 | **Поведенческий рычаг** | Какая автопилотная привычка находится в одном повороте от дохода? |

Полная методология в [references/signal-extraction-framework.md](references/signal-extraction-framework.md).

## Подробная установка

### Шаг 1 — Клонируйте репозиторий

```bash
git clone https://github.com/realteamprinz/midas-skill
cd midas-skill
```

### Шаг 2 — Поместите там, где ищет Claude Code

Для Claude Code скиллы живут в `.claude/skills/` в **корне git-репозитория** (на проект) или в `~/.claude/skills/` (глобально). Выберите один вариант:

```bash
# На проект: сначала cd в git-репозиторий, к которому хотите привязать скилл
mkdir -p .claude/skills
cp -r /path/to/midas-skill .claude/skills/midas-skill

# Или глобально (доступно в каждом проекте)
mkdir -p ~/.claude/skills
cp -r /path/to/midas-skill ~/.claude/skills/midas-skill
```

### Шаг 3 — Проверьте, что скилл загрузился

Откройте сессию Claude Code в репозитории, куда вы его установили. Введите:

```
What skills are available?
```

Midas должен появиться как `midas-skill`. Если нет, проверьте, что `SKILL.md` находится в корне установленной папки, а YAML frontmatter валиден.

### Шаг 4 — Запустите первый майнинг

Вставьте любой шумный ввод с триггерной фразой:

```
Midas, mine this — последние 50 сообщений из моего #general Slack:
[вставить содержимое]
```

Midas создаст ваш первый Сигнальный отчёт и добавит первую запись в `evolution/evolution.jsonl`.

### Шаг 5 — Скармливайте разнообразие со временем

Midas становится острее с более разнообразным вводом. См. [references/sources-guide.md](references/sources-guide.md) для рекомендуемой недельной ротации (Slack в понедельник, фото в среду, браузер в четверг и т. д.). День 1 даёт догадки. День 30 даёт уверенность. День 90 даёт плейбук.

## Функции

### 🔍 6-линзовый движок извлечения

Каждый ввод проходит через Денежный сигнал / Пробел спроса / Арбитраж / Мост навыков / Сетевой путь / Поведенческий рычаг. Каждый сигнал отслеживается до точной фразы, изображения или точки данных, которая его произвела.

### 🧠 Самообучающийся кумулятивный интеллект

Midas не сбрасывается между сессиями. Каждый ввод добавляется в `evolution/evolution.jsonl`. Новые сигналы перекрёстно сверяются со всеми предыдущими. **Уверенность возрастает только через независимые перекрёстно сверенные доказательства.**

| Доказательства | Уверенность | Флаг Midas |
|---|---|---|
| 1 сигнал из одного источника | 15–35% | Наблюдаем |
| 2 независимых источника подтверждают | 40–65% | Рабочая гипотеза |
| 3+ независимых источника сходятся | 70–90% | 🏆 ЗОЛОТО — ДЕЙСТВУЙ |
| Прямая рыночная валидация | 85–95% | 🏆 ЗОЛОТО — ДЕЙСТВУЙ СЕЙЧАС |

### 🎯 Матчер паттернов знаменитых OS богатства

Предварительно построенные операционные системы богатства для [Маска](references/famous-models/elon-musk.md), [Баффета](references/famous-models/warren-buffett.md) и [Тиля](references/famous-models/peter-thiel.md). Когда ваши сигналы накапливаются, Midas автоматически проверяет, какой плейбук структурно соответствует вашей ситуации.

> «Ваш паттерн похож на раннего Тиля: монополия в нише, которую никто не воспринимает всерьёз. Плейбук Тиля гласит: сначала доминируйте на маленьком рынке.»

### 💼 Анализатор сделок

Направьте Midas на любую конкретную транзакцию (свою или публичную M&A) с помощью *"Midas, analyze this deal"* и получите структурированную разбивку из 8 частей: стороны, структура, источник средств, распределение риска, путь выхода, скрытый рычаг, вердикт, совпадение паттерна. См. [references/deal-template.md](references/deal-template.md).

### ⚡ Действие, а не анализ

Каждый Сигнальный отчёт заканчивается **Немедленным следующим действием** — конкретным глаголом + конкретным существительным. Никогда «изучить варианты». Всегда «написать Дэйву сегодня вечером и спросить, сколько часов в неделю он тратит на ручной отчёт».

## Принципы дизайна

1. **Ваш шум — это ваша руда.** Каждое обыденное взаимодействие содержит потенциальные сигналы богатства.
2. **Повторение — главный сигнал.** Если что-то появляется дважды в вашей жизни — это рынок.
3. **Перекрёстная проверка побеждает единственный источник.** Один сигнал — догадка. Три — убеждение.
4. **Действие над анализом.** Midas выдаёт следующие шаги, а не эссе.
5. **Сложный интеллект.** День 1 даёт догадки. День 90 даёт плейбук.
6. **Без моральных суждений.** Midas изучает, как движутся деньги, а не как они *должны* двигаться.
7. **Честные границы.** Никаких прогнозов. Никаких гарантий. Никаких финансовых советов.
8. **Только публичная информация.** Только то, что вы добровольно скармливаете.
